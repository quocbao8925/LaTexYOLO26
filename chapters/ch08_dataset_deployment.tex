\subsection{Format nhãn YOLO}

Mỗi ảnh có file \texttt{.txt} tương ứng:
\begin{verbatim}
cls  cx  cy  w  h
0  0.5  0.6  0.3  0.8
\end{verbatim}
Tọa độ normalized $[0, 1]$, $(cx, cy)$ = tâm box, $(w, h)$ = kích thước.

\subsection{Tiền xử lý dữ liệu}

\begin{enumerate}
    \item \textbf{Letterbox}: Resize giữ tỷ lệ, pad pixel $(114, 114, 114)$
    \item \textbf{Normalize}: Chia 255, chuyển float32
    \item \textbf{xywh $\to$ xyxy}: Chuyển đổi format box khi tính loss
\end{enumerate}

\section{Triển khai trên Raspberry Pi 5}

\subsection{Pipeline Export}

\begin{enumerate}
    \item \textbf{Strip O2M}: Loại bỏ O2M head (cv2, cv3), chỉ giữ O2O
    \item \textbf{FP16 Quantization}: Chuyển weights sang FP16
    \item \textbf{PyTorch $\to$ ONNX}: Export model graph
    \item \textbf{ONNX $\to$ NCNN}: Chuyển đổi sang format NCNN (.param + .bin)
\end{enumerate}

\subsection{NCNN Runtime}

NCNN~\cite{ncnn} tối ưu cho ARM:
\begin{itemize}
    \item Viết bằng C++, zero dependency
    \item Tối ưu NEON/SIMD cho CPU ARM Cortex-A76
    \item Dung lượng nhỏ, khởi động nhanh
\end{itemize}

\section{Inference Pipeline}

\begin{enumerate}
    \item Camera capture frame (MIPI CSI hoặc USB)
    \item Letterbox resize $\to$ $320 \times 320$
    \item NCNN forward pass (O2O head only)
    \item Decode: LTRB $\to$ xyxy $\times$ stride
    \item Top-K selection ($K=300$) theo confidence
    \item Output: $(x_1, y_1, x_2, y_2, \text{score}, \text{class})$
\end{enumerate}

Không cần NMS --- O2O head đã tạo dự đoán duy nhất cho mỗi đối tượng.
